## 一、什么是大运和流年？

### 大运（大限）

**定义**：每十年为一个周期，代表这十年的整体运势主题

**作用**：
- 看人生不同阶段的重心
- 判断十年整体吉凶
- 分析重大人生转折

**排列方式**：
- 阳男阴女：顺行（顺时针）
- 阴男阳女：逆行（逆时针）

### 流年

**定义**：每年的运势，代表当年的具体事件

**作用**：
- 看当年的吉凶
- 分析具体事件
- 选择时机做决策

**排列方式**：每年移动一个宫位

## 二、如何确定大运？

### 1. 确定起运岁数

起运岁数由**五行局**决定：

| 五行局 | 起运岁数 |
|--------|----------|
| 水二局 | 2岁起运 |
| 木三局 | 3岁起运 |
| 金四局 | 4岁起运 |
| 土五局 | 5岁起运 |
| 火六局 | 6岁起运 |

**举例**：
- 水二局的人，2岁起第一个大运
- 木三局的人，3岁起第一个大运

### 2. 确定大运宫位

从命宫开始，按照顺行或逆行的方向，每十年移动一个宫位。

**阳男阴女（顺行）**：
```
命宫(1-10岁) → 父母宫(11-20岁) → 福德宫(21-30岁) → 田宅宫(31-40岁) → ...
```

**阴男阳女（逆行）**：
```
命宫(1-10岁) → 兄弟宫(11-20岁) → 夫妻宫(21-30岁) → 子女宫(31-40岁) → ...
```

## 三、如何分析大运？

### 1. 看大运宫位的星曜

大运进入某个宫位，该宫位的星曜会被激活。

**分析方法**：
- 看大运宫位的主星
- 看辅星和煞星
- 看三方四正

**举例**：
某人33-42岁大运进入田宅宫，田宅宫有天府、禄存：
- 这十年适合置业、买房
- 家庭稳定，有积蓄
- 事业基础稳固
