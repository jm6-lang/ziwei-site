## 使用说明

1. **输入阳历（公历）出生日期**、出生时辰、性别
2. **生成命盘**：系统自动计算十四主星落宫
3. **查看分析**：命盘总览、过去十年运势回顾、未来十年预测

## 排盘原理

紫微斗数以出生时的年、月、日、时为依据：

- **定盘**：根据出生时辰确定十二宫位
- **安星**：根据六十甲子纳音安十四主星
- **大运**：每十年一运，顺逆由阴阳男女决定
- **流年**：每年一运，与太岁相配
